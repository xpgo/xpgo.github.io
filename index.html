Selected Self-hosted Services

Personal data are our privacy and treasure, so it would be better to store all your privat data on your own server. Here are some recomented self-hosted services I am using and feels great. Give them a try.

Simpledash

A simple, fully responsive Dashboard to forward to the services of your choice!

Shaarli

The personal, minimalist, super-fast, database free, bookmarking service.

Chevereto

Allows you to create a beautiful and full-featured image hosting website on your own server.

Gitlab

Manage Git repositories with fine grained access controls that keep your code secure

Leanote

Not just a notepad! Note-taking made easy: Leanote incorporates a clean and intuitive interface.

Shiori

A simple bookmarks manager written in Go language. Intended as a simple clone of Pocket.

Tiny Tiny RSS

Tiny Tiny RSS is a free and open source web-based news feed (RSS/Atom) reader and aggregator.

Jekyll

A static web generator I am using to manage my github pages. Use markdown files for blog publish.
